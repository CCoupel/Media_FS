# Architecture

## Package layout

```
cmd/mediafs/          Entry point — CLI parsing, tray launch
internal/
  config/             Config struct, YAML load/save, server registry
  connector/          MediaConnector interface + per-server implementations
    jellyfin/         Jellyfin REST connector (also covers Emby with minor adapter)
    emby/             Emby adapter (wraps jellyfin connector, overrides auth headers)
    plex/             Plex connector (P3)
  vfs/                WinFSP/FUSE filesystem — maps VFS calls to connector calls
  cache/              SQLite metadata cache with per-type TTLs
  downloader/         Multi-chunk parallel HTTP downloader (used for local copy)
  tray/               System tray icon + menu (fyne-io/systray)
  webui/              Local HTTP config UI, opened in browser from tray
pkg/
  nfo/                NFO XML generation (Kodi-compatible)
assets/
  icons/              .ico files per connector type (embedded in binary)
  tray/               Tray icon variants (idle, active, error)
```

## Data flow — directory listing

```
Explorer readdir("Z:\cyril@HomeServer\Films")
  → vfs.ReadDirectory()
    → cache.GetItems(serverID, parentID)   ← cache hit: return immediately
    → [miss] connector.GetItems()          ← HTTP call to Jellyfin API
      → cache.StoreItems()
    → build []FileInfo (name, size, isDir)
```

## Data flow — file read (streaming)

```
VLC opens "Z:\cyril@HomeServer\Films\Inception (2010)\Inception.mkv"
  → vfs.Open() → returns handle with streamURL
  → VLC issues Read(offset, length) via range request
    → vfs.Read(handle, offset, buf)
      → http.Get(streamURL, Range: bytes=offset-end)
      → copy to buf
```

## Data flow — file copy (parallel download)

```
Explorer copies Inception.mkv to D:\
  → OS issues sequential Read() calls for the full file
  → vfs detects full-file read pattern → delegates to downloader
    → downloader splits file into N chunks
    → N goroutines fetch chunks in parallel (HTTP Range)
    → chunks reassembled in order → returned to OS
```

## Virtual files (generated in-memory, never written to server)

| File | Generated by |
|---|---|
| `*.nfo` | `pkg/nfo` — XML from item metadata |
| `poster.jpg`, `fanart.jpg` | Proxied from connector.GetArtworkURL() |
| `desktop.ini` | `internal/vfs` — sets folder icon per connector type |

## Config + tray interaction

```
mediafs starts
  → config.Load() reads config.yaml
  → tray.Start() shows icon in system tray
  → for each enabled server: connector.Connect() + vfs.Mount()
  → tray menu: status per server, "Configuration" opens localhost webui
  → webui writes config.yaml on save → triggers hot-reload
```

## TTL cache (SQLite)

| Table | TTL | Notes |
|---|---|---|
| `items` | 5 min | Directory listings |
| `metadata` | 1 hour | Per-item details |
| `artwork` | 24 hours | Stored as blobs |
| `nfo` | 1 hour | Generated XML |
